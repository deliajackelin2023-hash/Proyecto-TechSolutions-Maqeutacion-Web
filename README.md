#  Proyecto TechSolutions

###  Descripción del Proyecto
<p>
TechSolutions es un sitio web corporativo diseñado para representar la identidad digital de una empresa especializada en soluciones tecnológicas. 

El proyecto presenta una interfaz moderna, intuitiva y adaptable a diferentes dispositivos, permitiendo mostrar información institucional, servicios, portafolio, equipo de trabajo y canales de contacto de manera profesional.
</p>



![](https://i.postimg.cc/HLqKQGZ7/logodetecsolutions.png) 

### Objetivos
- Desarrollar un sitio web moderno y responsive.
- Mejorar la presencia digital de empresas tecnológicas.
- Aplicar buenas prácticas de desarrollo Front-End.
- Brindar una experiencia de usuario intuitiva y atractiva.

### Características
- Diseño Responsive.
- Interfaz moderna.
- Navegación intuitiva.
- Carrusel interactivo.
- Modo Claro/Oscuro.
- Animaciones al desplazarse.
- Portafolio con filtros.
- Lightbox para imágenes.
- Validación del formulario de contacto.
- Código organizado y mantenible.

Tecnologías utilizadas

    Html5
    Css
    Bootstrap 5
	Bootstrap Icons
	Google Fonts
    JavaScript
